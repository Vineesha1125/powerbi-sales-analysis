# Retail Sales Analytics Dashboard

## Project Overview
Interactive Power BI dashboard analyzing retail sales performance across multiple dimensions including product categories, regions, and customer segments.

## Dataset
- Source: Sample Superstore dataset
- Records: 9,994 transactions
- Period: 2014-2017 (if dates were included)
- Categories: 3 (Furniture, Office Supplies, Technology)
- Sub-categories: 17
- Regions: 4 (Central, East, South, West)

## Key Metrics (KPIs)
- **Total Sales:** $2.30M
- **Total Profit:** $286.40K
- **Profit Margin:** 12.5%
- **Total Orders:** 9,994
- **Total Quantity:** 38K units

## Key Insights
1. **Top Category:** Technology generates the highest sales
2. **Regional Performance:** West region leads in sales distribution
3. **Product Analysis:** Phones, Chairs, and Storage are top-selling sub-categories
4. **Customer Segments:** Analysis across Consumer, Corporate, and Home Office

## Technical Skills Demonstrated
- **Data Preparation:** Power Query data cleaning and transformation
- **Data Modeling:** Created custom calculated columns (Profit Margin %, Order Size, IsProfitable)
- **DAX Measures:** Built 8 measures including aggregations and calculations
- **Visualizations:** 
  - KPI Cards for key metrics
  - Bar charts for categorical analysis
  - Donut chart for distribution analysis
  - Data tables for detailed breakdowns
- **Interactivity:** Cross-filtering enabled across all visuals
- **Design:** Clean, professional dashboard layout

## Tools Used
- Power BI Desktop
- Power Query Editor
- DAX (Data Analysis Expressions)

## Business Value
This dashboard enables stakeholders to:
- Monitor sales performance at a glance
- Identify top-performing products and categories
- Analyze regional sales distribution
- Track profitability metrics
- Make data-driven business decisions
